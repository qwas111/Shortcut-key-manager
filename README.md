# 快捷键管理器 (Shortcut Manager)

一个基于 PyQt5 的现代化快捷键管理软件，具有科技风格的界面设计。允许用户自定义全局快捷键来快速启动应用程序或打开网址。

##项目地址
[官网](https://nanet.ct.ws/md?f=skm)
[GitHub](https://github.com/qwas111/Shortcut-key-manager)

## 功能特性

### ✨ 核心功能
- 🎯 **全局快捷键管理**：自定义全局快捷键组合
- 🚀 **快速启动**：通过快捷键快速启动应用程序或打开网址
- 🎨 **现代化界面**：深色科技风格的用户界面
- 🗂️ **配置管理**：自动保存和加载快捷键配置
- 🖥️ **系统集成**：支持开机自启动

### 📊 高级特性
- 🔧 **实时配置**：添加、编辑、删除快捷键无需重启
- 📋 **快捷键捕获**：可视化快捷键捕获界面
- 🎮 **系统托盘**：最小化到系统托盘，后台运行
- 🔄 **热重载**：修改快捷键配置后自动重新注册
- 📁 **独立配置**：使用 JSON 文件存储配置，易于备份

## 安装指南

### 环境要求
- Python 3.7 或更高版本
- Windows 操作系统
- 推荐使用虚拟环境
